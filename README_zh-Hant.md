# mFirewall

[English](README.md) | [简体中文](README_zh-Hans.md)

mFirewall 是免費的開源 macOS 防火牆，用於監控並阻止未經授權的傳出網路連線。

完整的安裝、使用說明和常見問題請參閱[使用者指南](docs/USAGE.md)（英文），也發布於 <https://lauriat.fr/outils/mfirewall>。
